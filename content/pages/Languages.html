Programming Languages for Data Science

There are many programming languages used in Data Science, including

High Level Languages

  • R
  • Python
  • MATLAB
  • Julia
  • JavaScript (primarily in visualization, dashboards and mashups)
  • SAS

Lower-level Languages

  • C++
  • C
  • Java
  • FORTRAN
They all have important, but quite different, roles, and have various advantages and disadvantages.

Domain-specific languages

  • UNIX shell
  • Regular expressions
  • SQL
  • Pig
  • XPath
